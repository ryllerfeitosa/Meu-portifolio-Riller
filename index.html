Máquina de Garra Automatizada (TCC)

Desenvolvimento de uma máquina de pegar bichinhos inspirada em modelos comerciais utilizados em shoppings, integrando conceitos de automação, eletrônica e sistemas embarcados. O projeto utiliza Arduino Mega com programação em C++ para controle de motores, lógica de movimentação e interação com o usuário.

Destaques Técnicos:
- Simulação de circuitos e validação de lógica utilizando Proteus.
- Modelagem mecânica da estrutura e componentes com Autodesk Inventor.
- Desenvolvimento e testes de firmware na Arduino IDE.
- Integração entre sistemas elétricos, mecânicos e controle lógico.

API Estados & Cidades

Desenvolvimento de uma API RESTful utilizando Node.js e Express. O projeto realiza o processamento de dados geográficos do Brasil, permitindo filtros por siglas, regiões e capitais. Demonstra domínio em integração Back-end e manipulação de JSON.

API Movie Manager & Controle de Catálogo

Desenvolvimento de uma API RESTful escalável em Node.js para o gerenciamento de catálogos cinematográficos. O sistema implementa o padrão MVC (Model-View-Controller) e DAO, garantindo uma separação clara entre a lógica de negócio e a persistência de dados em MySQL.

Destaques Técnicos:
- Integração de Query Builder Knex.js para operações assíncronas de CRUD com alta performance.
- Implementação de segurança CORS e controle de Content-Type para garantir integridade nas requisições JSON.
- Validação rigorosa de Business Rules para tratamento de campos obrigatórios e tipos de dados.
- Modelagem de dados estruturada com scripts SQL otimizados e suporte a avaliações e precificação dinâmica.
